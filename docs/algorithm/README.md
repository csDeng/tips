# 简介

本页主要是我学习数据结构与算法的打卡点。

:::details 代码技巧

1. 架构优于逻辑;
2. 工程化，解耦合;
3. 代码可读性优于代码量;


    * 使用含义清晰的变量名
    * 使用简单易读的处理逻辑


4. 注重`code review`;
5. 模块化开发；
6. 缩进层数尽量少于3；

    * 因为代码的缩进，会增加代码的理解难度。


:::
